Wie ist die App aufgebaut?

Alles ist hierarchisch organisiert – von der grössten Einheit (Zone) bis zum einzelnen Ding. Du kannst so spezifisch sein wie du willst: ein Ding kann direkt in einem Raum liegen, oder tief verschachtelt in einem Behälter in einem Regalfach.

Zone

Oberste Ebene. Gruppiert mehrere Räume.

Beispiel: Keller, Erdgeschoss, Estrich

Raum

Ein physischer Raum innerhalb einer Zone.

Beispiel: Werkstatt, Garage, Schlafzimmer

Gestell

Ein Möbelstück oder eine Ablage im Raum.

Beispiel: Werkbank, Ivar-Regal, Werkstattwagen, Kallax

Regalfach

Eine Unterteilung innerhalb eines Gestells. Optional.

Beispiel: Kallax A1, Kallax B3, Schublade 2

Behälter

Eine Box, Kiste oder Schublade mit Inhalt. Optional.

Beispiel: Kuggis «Elektro», Schublade «Zangen»

Ding

Der eigentliche Gegenstand. Kann überall sitzen.

Beispiel: Akku-Bohrer, Camping-Matte, Gitarrenkoffer

Wie flexibel kann ein Ding platziert werden?

Situation Standort-Pfad
Ding liegt einfach irgendwo im Raum Raum
Ding liegt auf einem Regal (kein Fach, kein Behälter) Raum → Gestell
Ding liegt in einem Behälter direkt auf dem Regal Raum → Gestell → Behälter
Ding liegt in einem Kallax-Fach (ohne Behälter) Raum → Gestell → Regalfach
Ding liegt in einer Box in einem Kallax-Fach Raum → Gestell → Regalfach → Behälter
{# ── Erste Schritte ───────────────────────────────────────────────────────── #}

Erste Schritte

  1. Zone anlegen – z.B. «Keller» → Standorte
  2. Räume anlegen – z.B. «Werkstatt», «Garage»
  3. Gestelle anlegen – z.B. «Werkbank», «Kallax»
  4. Optional: Regalfächer anlegen – z.B. für Kallax-Zellen
  5. Optional: Behälter anlegen – z.B. Kuggis, Schubladen
  6. Dinge erfassen – und dem richtigen Standort zuweisen → Standorte
